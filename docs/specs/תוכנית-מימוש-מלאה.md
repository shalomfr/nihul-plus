## שלב 4: מנוע אוטומציות (הלב)
### קבצים:
- `app/src/lib/automation/engine.ts` - Workflow engine core
- `app/src/lib/automation/triggers.ts` - Trigger matching logic
- `app/src/lib/automation/actions.ts` - Action executors (email, create-doc, update-record, notification)
- `app/src/lib/automation/templates.ts` - 10 pre-built workflow templates
- `app/src/lib/automation/scheduler.ts` - Cron-based scheduled tasks

### ארכיטקטורה:
```
Event Published → Event Bus → Engine.matchWorkflows()
  → For each matching workflow:
    → Evaluate conditions (if/else rules)
    → Queue actions via BullMQ
    → Worker picks up job → executes actions in order
    → Log result to WorkflowExecution
```

### Action Types:
1. `SEND_EMAIL` - שליחת מייל (Nodemailer)
2. `CREATE_DOCUMENT` - יצירת מסמך (PDF/record)
3. `UPDATE_RECORD` - עדכון רשומה בDB
4. `SEND_NOTIFICATION` - התראה בממשק
5. `WEBHOOK` - קריאת HTTP (לשימוש עתידי)

### 10 תבניות:
1. קבלה אוטומטית לתרומה
2. תזכורת ניהול תקין (30 יום)
3. סיכום ישיבת ועד
4. דוח חודשי אוטומטי
5. תזכורת דוח שנתי
6. ברכות חג לתורמים
7. אישור שעות מתנדבים
8. התראת חריגה מתקציב
9. גיבוי מסמכים
10. בדיקת סטטוס רשם עמותות
